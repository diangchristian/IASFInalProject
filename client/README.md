# Login System (React + TypeScript + Vite)

This project is migrated to TypeScript while preserving the existing UI design and Tailwind-based styling.

## Scripts

- `npm run dev` - Start development server
- `npm run lint` - Run ESLint
- `npm run typecheck` - Run TypeScript checks
- `npm run build` - Type-check and build production assets
- `npm run preview` - Preview production build

## Design Consistency Notes

- Existing Tailwind classes and component structure are preserved.
- Theme colors and typography remain in `tailwind.config.js`.
- Global styling remains in `src/index.css`.
